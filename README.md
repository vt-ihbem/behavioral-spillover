# behavioral-spillover
This is the repository for the NSF IHBEM Behavioral Spillover project. This repository contains the data presented and the code used for all simulations and analysis in the publication [1].

[1] LeJeune L, Ghaffarzadegan N, Childs LM, Saucedo O. The behavioural spillover effect: modelling behavioural interdependencies in multi-pathogen dynamics. Journal Of Biological Dynamics. Pending publication.
